## How to use

-- Keyboard Layout
```sh
$ source <(curl -Ls https://git.io/vxkiW)
$ sudo reboot
```

-- Show Miner Claymore (CTRL+A+D to detach screen)
```sh
$ source <(curl -Ls https://git.io/vxkiZ)
$ show miner
```

-- Update Timezone
```sh
$ source <(curl -Ls https://git.io/vxk6o)
```
